# Portfolio d'Adrien Ozon

Portfolio statique en HTML, CSS et JavaScript, concu pour une publication simple sur GitHub Pages.

## Lancer en local

Ouvrez `index.html` dans un navigateur ou servez le dossier avec un serveur statique.

## Structure

- `index.html` : structure de la page
- `styles.css` : design system et mise en page
- `script.js` : animations et interactions
- `assets/cv/adrien-ozon-cv.pdf` : CV telechargeable
